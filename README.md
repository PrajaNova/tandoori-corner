# Tandoori Corner

Pnpm workspace for the Tandoori Corner web and backend services.

## Apps

- `apps/tc-web-app` - Next.js 16 App Router web app.
- `apps/rc-backend` - Fastify TypeScript backend for menu, order, and restaurant services.

## Commands

- `pnpm dev:web` - run the web app.
- `pnpm dev:backend` - run the backend API.
- `pnpm build` - build all workspace packages.
- `pnpm lint` - run Biome checks across the workspace.

Next.js app using TypeScript, Tailwind CSS, and Biome.

## Scripts

```bash
pnpm dev
pnpm build
pnpm lint
pnpm format
```
